PUZZLE 1 — The Lost Expedition
================================

Files:
  • puzzle1.html   — the puzzle page
  • verifier1.py  — answer checker
  • solution1.txt — organiser solution (keep hidden from participants!)

Participant Instructions:
  1. Open puzzle1.html in your browser.
  2. The page shows an explorer's journal with three fragments hidden in the source.
  3. Right-click → "View Page Source" or press F12 → Elements tab.
  4. Find all three fragments and combine them in the correct order.
  5. Enter your answer into the verifier:
       python verifier1.py

Hint for participants:
  "Three pieces are hidden across the page. Find them all and combine them in order."

Expected answer format: One word (no spaces), e.g. DRAGONFLY
Technique covered: HTML Comments, Hidden div elements (display:none)
Difficulty: ⭐ Beginner
