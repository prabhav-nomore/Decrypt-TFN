PUZZLE 9 — The Antique Shop
=============================

Files:
  • puzzle9.html   — the puzzle page
  • verifier9.py  — answer checker
  • solution9.txt — organiser solution (keep hidden from participants!)

Participant Instructions:
  1. Open puzzle9.html in your browser.
  2. Three antique items are displayed, each with a TAG label.
  3. Try hovering your mouse over each TAG to reveal tooltip text.
  4. OR open DevTools (F12) → Elements and inspect the title="" attribute 
     on each tag element.
  5. Find the safe code and verify:
       python verifier9.py

Hint for participants:
  "Every item has a tag. Tags carry more than just reference numbers — 
   hover over them, or inspect their title attributes."

Expected answer format: One word, e.g. OBSIDIAN
Technique covered: HTML title="" attribute (native browser tooltip)
