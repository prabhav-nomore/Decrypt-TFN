Image Cipher #4 — Green Room
================================

Files included:
  • cipher4.png   — the puzzle image
  • README_cipher4.txt — this file

Participant instructions:
  1) Open cipher4.png in an image editor that supports channel viewing
     (GIMP → Colors → Curves, Photoshop → Channels panel, or use Python/PIL)
  2) Isolate and view ONLY the Green (G) channel
  3) Boost the Green channel contrast to maximum
  4) The hidden message will appear

Hidden message format:  PASS: <WORD>

Hint: Red and Blue channels are noise. Green holds the truth.
