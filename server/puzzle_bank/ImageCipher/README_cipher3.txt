Image Cipher #3 — Negative Space
================================

Files included:
  • cipher3.png   — the puzzle image
  • README_cipher3.txt — this file

Participant instructions:
  1) Open cipher3.png in any image editor
  2) Apply the INVERT (or Negative) filter to the image
     (Photos app → Edit → … → Invert  |  GIMP → Colors → Invert)
  3) The hidden message will flip into visibility against the dark background

Hidden message format:  DECODE: <WORD>

Hint: What you see is white noise. What you don't see is the opposite.
