Image Cipher #9 — Shadow Exposure
================================

Files included:
  • cipher9.png          — the puzzle image
  • README_cipher9.txt   — this file

Participant instructions:
  1) Open cipher9.png in any image editor
  2) Dramatically increase EXPOSURE or SHADOWS slider
     (Photos app → Edit → Light → Shadows max, Exposure max)
     OR apply a strong Gamma correction (gamma < 0.2)
     OR use Curves: pull the dark end of the curve steeply upward
  3) The hidden message lurking in the near-black shadows will appear

Hidden message format:  GAMMA: <WORD>

Hint: The image says "TOO DARK TO READ." It's right — without the right
      exposure settings. Push your shadows slider all the way up.
