Image Cipher #7 — Upside Down
================================

Files included:
  • cipher7.png          — the puzzle image
  • README_cipher7.txt   — this file

Participant instructions:
  1) Open cipher7.png in any image editor
  2) Rotate the image 180° (flip it upside down)
     (Photos app → Edit → Rotate  |  GIMP → Image → Transform → Rotate 180°)
  3) Boost brightness/contrast if needed
  4) The hidden message is now readable at the top of the rotated image

Hidden message format:  TURN: <WORD>

Hint: The image says "NOTHING UP HERE." But what about down there?
      Rotate a full half-turn and look again.
