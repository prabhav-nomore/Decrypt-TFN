# 🔐 Progressive Caesar Cipher Challenge

## 📌 Overview

This challenge uses a **Progressive Caesar Cipher**, where the shift increases for each letter in the sentence.

---

## 🧠 How It Works

* Each letter is shifted forward based on its position:

  * 1st letter → **+1** (A → B)
  * 2nd letter → **+2** (B → D)
  * 5th letter → **+5** (C → H)
* The shift keeps increasing as you move ahead in the sentence.

---

## ⚠️ Rules

* Only **letters** are shifted
* **Spaces and punctuation remain unchanged**

---

## 🧩 Hints

* 🔍 Start with small words like **"the"**
* 🔢 Keep track of the increasing shift
* 🔄 If stuck, try reversing the process

---

## 🎯 Objective

Decode the given messages and find the original sentences.

---

Good luck! 🔓
