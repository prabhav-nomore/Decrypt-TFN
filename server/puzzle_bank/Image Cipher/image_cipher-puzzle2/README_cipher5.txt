Image Cipher #5 — Blue Horizon
================================

Files included:
  • cipher5.png          — the puzzle image
  • README_cipher5.txt   — this file

Participant instructions:
  1) Open cipher5.png in an image editor that supports channel viewing
     (GIMP → Colors → Curves, Photoshop → Channels panel, or use Python/PIL)
  2) Isolate and view ONLY the Blue (B) channel
  3) Boost the Blue channel contrast to maximum
  4) The hidden message will emerge

Hidden message format:  PASS: <WORD>

Hint: The sky looks clear. But every sky has a hidden depth.
      Red and Green channels are noise — only Blue holds the truth.
