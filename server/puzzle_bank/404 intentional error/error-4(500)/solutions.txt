Solutions (organizer reference) — Puzzle 500

Working files:
- index.html links to pages/process.html (loads normally, no error).

Example single-line edit participants should make:
1) Change link href:
   <a href="pages/process.html">Submit Data</a>
   -> <a href="pages/crash.html">Submit Data</a>

   Clicking the link sends GET /pages/crash.html to the server.
   The server responds with HTTP 500 Internal Server Error.
   The response body contains "Clue: 500".
   DevTools Network tab shows the 500 status on the request row.

Organizer tip:
- Run with python3 run_server.py (port 8003).
- The crash path list (CRASH_PATHS) in run_server.py can be extended if needed.
- To increase difficulty, remove the body clue so participants must use
  the DevTools Network tab to read the status code.
- To simulate a real crash feel, you can add a short time.sleep(2) before
  sending the 500 response to make it feel like the server is "hanging".
