Image Cipher #2 — Red Eye
================================

Files included:
  • cipher2.png   — the puzzle image
  • README_cipher2.txt — this file

Participant instructions:
  1) Open cipher2.png in an image editor that supports channel viewing
     (GIMP → Colors → Curves, Photoshop → Channels panel, or use Python/PIL)
  2) Isolate and view ONLY the Red (R) channel
  3) Boost the Red channel contrast to maximum
  4) The hidden message will appear

Hidden message format:  SECRET: <WORD>

Hint: The other two colour channels are a dead end.
