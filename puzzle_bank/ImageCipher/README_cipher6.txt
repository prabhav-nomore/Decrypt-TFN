Image Cipher #6 — Mirror Mirror
================================

Files included:
  • cipher6.png          — the puzzle image
  • README_cipher6.txt   — this file

Participant instructions:
  1) Open cipher6.png in any image editor
  2) Flip / Mirror the image HORIZONTALLY
     (Photos app → Edit → Flip  |  GIMP → Image → Transform → Flip Horizontally)
  3) Boost brightness/contrast if needed
  4) The hidden message will now read correctly

Hidden message format:  FLIP: <WORD>

Hint: The image says "WRONG DIRECTION." Take that literally.
      Something in the image is already showing you the answer — backwards.
